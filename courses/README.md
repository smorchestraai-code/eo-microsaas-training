# Courses

Structured training content for EO MicroSaaS students. Each subfolder is one course module.

## Planned courses

| # | Course | Status | Target audience |
|---|--------|:------:|-----------------|
| 01 | MicroSaaS Fundamentals | Planned | Brand-new to SaaS |
| 02 | Idea → Validated Niche | Planned | Have an idea, need to validate |
| 03 | Business Brain (positioning, ICP, offer) | Planned | Post-validation |
| 04 | GTM Asset Factory (pre-launch assets) | Planned | Ready to launch |
| 05 | Custom Skills for Your Niche | Planned | Advanced students |
| 06 | Architecture + Claude Code Handover | Planned | Ready to build |

## Structure per course

```
courses/NN-course-name/
├── README.md            # What this course covers, who it's for, time commitment
├── lessons/             # Individual lesson markdown files
│   ├── 01-intro.md
│   ├── 02-first-exercise.md
│   └── ...
├── exercises/           # Hands-on activities
├── templates/           # Any course-specific templates
└── examples/            # Worked examples
```
